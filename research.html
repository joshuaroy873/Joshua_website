Current Research Interests

  • Hardware-based Malware Detector: Application of debug and test hardware for Malware detection.

  • Functional Safety and Security: Functional testing for ensuring functional safety in automobiles.

  • Hardware Security: Verifying security vulnerabilities in next generation hardware.

  • Edge Computing: Developing efficient inference algorithm and securing them in the edge.

  • Neuromorphic Computing: Designing low-power neuromorphic computing hardware.

  • Post-Quantum Cryptography: Developing hardware primitives for Post-Quantum Cryptography algorithms.
